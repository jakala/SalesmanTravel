# SalesmanTravel
A solution of Salesman Travel
## requirements:
I'm using phpunit 6.3, so Php7 is mandatory.

`apt-get install phpunit`

## Install
`git clone https://github.com/jakala/SalesmanTravel.git`

## Run:
You can execute example by this line:

`php -f Solve.php`
